## 作用

术语可以降低沟通成本

## 常见术语
### web

web指互联网

### w3c

万维网联盟，非盈利性组织。为互联网提供各种标准。
官网：[w3.org](https://w3.org)

### XML

可扩展的标记语言（extension markup language），用于定义文档结构的。

例：请在每周周日12点，从爱奇艺下载最新动漫《海贼王》
```xml
<任务>
	<执行日期>每周日</执行日期>
	<执行时间>12:00</执行时间>
	<下载地址>爱奇艺</下载地址>
	<下载目标>海贼王</下载目标>
	<目标版本>最新版</目标版本>
<任务>
```
